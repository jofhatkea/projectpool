Smack The Fly

  4. If the fly is hit certain times, change the speed and/or the size of the fly.
  5. Spise up the game with other challenges to the user (e.g. add a seccond fly) and other variations...

Timer / stop watch

Create a custom made timer / stop watch.

The user can click either a start button to make the watch count seconds or some fixed "timer" buttons to count down from e.g. 5, 10 and 15 minutes.

It shoud be possible to reset the watch at any time.

Advanced features

  1. Make a pause / resume button (toggle pause/resume to the timer).
  2. Play a sound when the timer reaches 00.
  3. Play a sound e.g. every 5 minutes.
  4. Make an input field (e.g. type=number") to lat the user type in any number of minutes to count down.
  5. Make a display with minutes and seconds (the seconds should reset to 00 after 59 seconds and add 1 to the minutes).

Animated to-do list

Elaborate on the "to do" list from last week.

Add another list (ul element) to the “to do” list and label it “Done". Move elements from the “To do” list to the “Done” list when they are clicked.

Advanced features

  1. Add an animation to this transfer (Hint: you might need to clone the li element to make this work: let cln = li.cloneNode(true)).
  2. Make it possible to click an element on the “Done” list to move it back to the “To do” list.
  3. Add an input field + a button to let the user add new items to the "To do" list.
  4. Add an eventlistener to the newly added items so they can be clicked and moved as the existing items.
